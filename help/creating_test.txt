Чтобы создать тест, нужно создать файл с расширением .json.
Обязательные теги отмечены '*'
--------------------------------------------------
Структура теста описана ниже:
{
	"title": "название теста",
	"description": "описание теста",
	*"questions": [
		{...} - вопросы
	],
	"author": "автор теста",
	"class": "класс теста",
	"control": true/false (этот тест контрольный / нет),
	"mistakes": количество выходов из окна перед остановкой теста (тест останавливается при достижении указанного числа выходов), работает только когда "control": true
}
--------------------------------------------------
Структура вопроса:
{
	*"type": тип вопроса,
	*"title": "название вопроса",
	"max_points": максимальное число очков за правильный ответ
	... - данные вопроса
}
--------------------------------------------------
Структура данных вопроса с ответом в виде строки:
*"type": 0,
*"answers": [
	"..." (верные варианты ответа)
],
"input-type": "Number" (тип ввода - число) | "Text" (тип ввода - текст) | "Fraction" (тип ввода - дробь вида x/y) | "Date" (тип ввода - дата) | "Time" (тип ввода - время)
--------------------------------------------------
Структура данных вопроса с галочкой:
*"type": 1,
*"variants": [
	"..." (варианты ответа)
],
*"rights": [
    ... (номера правильных ответов)
],
"check-rule": "AAR" (проверка - даёт максимальные баллы, если все ответы верны) | "ACC" (проверка - даёт баллы соответственно с правильными ответами) | "RIW" (проверка - уменьшает баллы за неправильный ответ)
--------------------------------------------------
Структура данных вопроса на сопоставление:
*"type": 2,
*"column1": [
    "..." (строки в первом столбце)
],
*"column2": [
    "..." (строки во втором столбце)
],
*"compares": [
    [номер строки первого столбца, номер строки второго столбца] (верные пары первого и второго столбцов)
],
"check-rule": "AAR" (проверка - даёт максимальные баллы, если все ответы верны) | "ACC" (проверка - даёт баллы соответственно с правильными ответами) | "RIW" (проверка - уменьшает баллы за неправильный ответ)
